研究与技术
AI 辅助 mRNA 疫苗设计：从 AlphaFold 到临床试验 生物医疗

UNSW RNA 研究所利用 ChatGPT（文献分析和方案设计）、AlphaFold（蛋白质结构预测）和 Grok（数据交叉验证）组合设计了个性化 mRNA 癌症疫苗，在犬类试验中实现肿瘤体积缩小 75%。

技术点评：这个案例的技术路径值得关注——研究者没有使用专门的药物发现 AI 平台（如 Insilico Medicine 的 Chemistry42），而是组合使用了通用 LLM 和开源生物信息学工具。这种"通用AI + 专业工具"的组合模式可能比端到端的专用系统更灵活，尤其在小规模个性化医疗场景中。

LATENT 系统：人形机器人实时网球对拉 机器人

清华/北大联合团队的 LATENT 系统让 Unitree G1 机器人实现了实时网球对拉。关键数据：仅需 5 小时人类运动数据训练，模拟环境正手成功率 96%。

技术点评：LATENT 的创新在于将 latent space 动作表示与实时反馈控制相结合。传统机器人运动控制需要大量的动力学建模，而 LATENT 通过在隐空间中学习运动模式，大幅降低了数据需求。5 小时数据即可训练的效率意味着快速适应新运动场景成为可能。

MIT Wi-Fi 3D 重建：非视觉感知的新范式 感知

MIT 利用生成式 AI 从 Wi-Fi 信号反射重建 3D 物体。这一技术将 RF 感知与 3D 重建结合，为机器人在低可见度环境（烟雾、黑暗、遮挡）中的操作提供了替代视觉方案。

技术点评：该方法利用 Wi-Fi 信号的多径效应（multipath propagation）作为隐式的空间采样，再通过生成模型将稀疏的 RF 特征映射到 3D mesh。与 LiDAR 或深度相机相比，Wi-Fi 感知的优势在于穿透性和低成本，但空间分辨率仍然是主要瓶颈。

WordPress AI Agent 自主发布功能 Agent

技术点评：这是 AI Agent "写入权限"扩展的又一案例。从只读（搜索、分析）到只写（发消息、发帖），Agent 的权限边界正在快速扩展。安全问题在于——如何确保 Agent 发布的内容符合平台政策和用户意图，避免"幻觉内容"被自动发布到公开网站。

"Tokenmaxxing"：AI 使用量成新 KPI 职场

纽约时报报道了"Tokenmaxxing"现象——公司将员工 AI 使用量（token 消耗、工具调用次数等）纳入绩效考核排行榜。

技术点评：从技术管理角度看，raw token 消耗并不是衡量 AI 效率的好指标。更有意义的指标应该是"AI 辅助任务完成率"或"人工干预频率"。过度追求 token 消耗量可能导致低效的 prompt 设计和不必要的 API 调用，反而增加成本而非提升生产力。

政策与版权
百科全书 vs OpenAI：RAG 时代的版权新战场 版权

大英百科全书和韦氏词典起诉 OpenAI 未经授权抓取约 10 万篇文章。诉讼特别针对 RAG（检索增强生成）工作流中的版权问题——即使模型不在训练中使用原文，在推理时检索并引用版权内容是否构成侵权？

技术点评：这起诉讼可能重新定义 RAG 系统的法律边界。当前 RAG 实现中的引用归属（attribution）机制通常不够精确，难以满足版权法的要求。行业可能需要开发更细粒度的内容溯源技术，如基于 content hash 的引用追踪系统。

技术点评：这对 AI 训练数据供应链产生直接影响。欧盟的立场（允许 opt-out）和英国的转向（要求 opt-in）之间的差异，正在形成一个全球 AI 训练数据合规的"拼凑式"监管图景。AI 公司可能需要建立区域化的训练数据管理系统。

LATENT：人形机器人实时动态运动控制 具身智能

清华/北大团队 LATENT 系统实现 Unitree G1 实时网球对拉。5h 数据训练，96% 模拟正手成功率。

  • 架构：LATENT（Latent Action Trajectory Encoding for Neural Transfer）采用 VAE 将人类运动轨迹编码到连续隐空间，再通过 RL policy 在隐空间中搜索最优动作序列。这避免了直接在高维关节空间中学习策略的维度灾难
  • 关键创新：motion retargeting 从人类到 G1 机器人的运动学映射不是简单的关节对应，而是通过 task-space 目标（球拍位置/速度/角度）进行重定向，允许机器人用不同的关节配置达到相同的击球效果
  • 数据效率：5 小时训练数据 vs 传统 RL 方法通常需要数百小时模拟数据。隐空间的正则化约束有效防止了小数据集上的过拟合
  • 实际部署挑战：sim-to-real gap 仍然是主要瓶颈。论文的 96% 成功率是模拟环境数据，真实环境中的成功率受地面摩擦、风阻、视觉延迟等因素影响预计会显著下降

  • 技术路线：CSI 采集 → RF feature extraction (CNN) → latent space encoding → conditional diffusion model → 3D mesh generation
  • 性能特征：空间分辨率 ~5cm（远低于 LiDAR 的 mm 级），但具有穿墙和非视距感知能力。适用于烟雾、完全黑暗等视觉系统失效场景
  • 应用方向：仓储物流机器人（在高密度货架间导航）、灾后搜救（烟尘环境）、安防监控（隐私友好的活动检测，无需摄像头）
