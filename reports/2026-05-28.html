融资与行业格局

Anthropic 以超 9000 亿美元估值完成 300 亿融资，成全球最贵 AI 公司 融资

Claude 的缔造者 Anthropic 本周正式完成了超过 300 亿美元的新一轮融资，估值突破 9000 亿美元，一举超越 OpenAI 在今年 3 月创下的 8520 亿美元纪录，成为全球估值最高的 AI 创业公司。

这笔资金背后站着 Sequoia、Microsoft、NVIDIA 等一众大牌投资人。更惊人的是 Anthropic 的收入增长曲线：从 2024 年 1 月的 8700 万美元年化收入，到现在年化 300 亿，16 个月暴涨 345 倍。预计最快今年 10 月 IPO。

AI 程序员 Devin 的公司 Cognition 融了 10 亿美元 融资

做出"AI 软件工程师 Devin"的 Cognition AI 刚刚完成 10 亿美元融资，估值飙升至 260 亿美元，8 个月内翻了一倍多。奔驰、NASA、高盛都在用 Devin 写代码。

Cognition 的年化收入从去年 5 月的 3700 万美元跳涨到如今的 4.92 亿美元，企业客户数连续 6 个月每月增长 50%。AI 编程赛道已经不是概念，而是真金白银的大生意。

SpaceX IPO 定档 6 月 12 日，目标史上最大 IPO

马斯克的 SpaceX 终于要上市了。投资者路演 6 月 4 日启动，6 月 11 日定价，6 月 12 日在纳斯达克以 SPCX 代码开始交易。目标估值 1.75 万亿美元，拟募资 750 亿美元，将成为人类历史上最大的 IPO。

SpaceX 旗下 Starlink 在 164 个国家拥有 1030 万订阅用户，已实现盈利。而整合了 xAI 的 SpaceXAI 部门 Q1 收入 8.18 亿美元，但仍在亏损。

模型与产品

NVIDIA 发布 Nemotron 3 Nano Omni：给 AI Agent 的全能眼耳嘴 模型

NVIDIA 发布了一个超级全能的开源 AI 模型 Nemotron 3 Nano Omni，它能同时处理视觉、音频、图像和文字，而且效率比同类模型高 9 倍。它专门为 AI Agent 设计，在文档理解、视频分析、音频理解六项排行榜上名列前茅。

这个模型的特别之处在于它既能跑在云端的大型服务器上，也能跑在边缘设备和手机上，让 AI Agent 真正拥有了"看、听、说"的能力。

DeepSeek V4-Pro 永久降价 75%，AI 价格战白热化 降价

DeepSeek 宣布从 5 月 31 日起，旗舰模型 V4-Pro 的 API 价格将永久降至原来的四分之一。缓存命中价格低至每百万 token 0.025 元，几乎接近免费。这让 V4-Pro 成为全球性价比最高的旗舰级模型。

与此同时，DeepSeek 完成了创纪录的 500 亿元人民币首轮融资，创始人梁文锋个人出资 200 亿。V4.1 版本定档 6 月发布，将支持原生 MCP 协议和多模态能力。

阿里 Qwen3.7-Max 登顶国产大模型榜首 模型

阿里巴巴在云峰会上发布了新一代旗舰模型 Qwen3.7-Max，在 Arena 全球盲测排行榜上位居国产模型第一，超越了 Kimi K2.6、DeepSeek V4-Pro 和 GLM-5.1，直逼 GPT-5.5 和 Claude Opus 4.7。

这个模型支持 100 万 token 上下文，最厉害的是它能自主执行长达 35 小时的复杂任务，推理速度提升了约 10 倍。这标志着国产大模型从"追赶"到"紧贴"国际顶尖水平。

面壁智能开源中国首个国产芯片训练的超轻量大模型 开源

面壁智能联合清华大学开源了 BitCPM-CANN，这是中国第一个完全基于华为昇腾芯片训练的 1.58-bit 大模型。所谓 1.58-bit 意味着模型被极度压缩，仅需原来六分之一的内存，但保留了 90%-97% 的能力。

8B 参数版本可以直接在主流旗舰手机上运行，这对国产 AI 芯片生态和端侧部署来说是个重要的里程碑。

企业与 Agent

微软 AI Agent 学会了操作电脑和打电话 Agent

微软 Copilot Studio 两大功能正式上线：电脑操作 Agent（像人一样点击按钮、输入文字、浏览界面来完成任务）和实时语音 Agent（500 毫秒以内的响应延迟，可以像真人一样和客户打电话）。

和传统的 RPA 机器人不同，这些 Agent 能理解自然语言指令并自动适应界面变化。它们已经集成到 Dynamics 365 客服中心，企业可以直接用来做客服和 IT 支持。

Gemini API 发生重大变更，开发者需紧急迁移 API

Google 的 Gemini API 在 5 月 26 日进行了一次重大架构变更，旧版接口将在 6 月 8 日被彻底移除。所有还没有迁移到新接口的应用都会直接停止工作。如果你的产品在用 Gemini API，这是一个两周倒计时的硬性截止期。

微软 Build 2026 大会下周开幕 大会

微软 Build 2026 开发者大会将于 6 月 2-3 日在旧金山举行。预计将正式宣布 Azure AI Foundry 多模型支持（不再只绑 OpenAI，Anthropic Claude 也将正式入驻），以及 GitHub Copilot 多 Agent 协作功能。这标志着微软从"押注 OpenAI"转向"多方下注"的战略转变。

政策与地缘

中国扩大 AI 研究人员出境限制 政策

中国政府机构正在扩大对 AI 研究人员的海外出行限制，涵盖私营企业和学术机构。私企的 AI 研究人员出国需要获得政府审批，学术机构也在单独实施自己的限制措施。这反映了在中美 AI 竞争加剧背景下，人才和技术管控的进一步收紧。

Anthropic 首尔办公室正式开业 扩张

Anthropic 任命前 Snowflake 韩国总经理崔基泳为韩国代表董事，首尔办公室于 5 月 26 日正式开业。韩国用户使用 Claude 的频率是人口规模预期的 3.5 倍以上，且以技术和创意工作为主，显示了亚太市场对高端 AI 工具的巨大需求。

五角大楼测试替代 Anthropic Claude 的 AI 模型 军事

美国国防部正在积极测试 OpenAI、Google 和 xAI 的 Grok 模型，以替代 Anthropic 的 Claude。此前国防部长 Hegseth 在 3 月以"供应链风险"为由将 Anthropic 列入黑名单，原因是 Anthropic 拒绝允许其 AI 用于自主武器和大规模监控。目前 25 名"核心用户"正在评估替代方案。

研究与社会

10 万人研究：AI 创造力已超过普通人 研究

一项涉及超过 10 万人的大规模研究发现，生成式 AI 在标准化创造力测试中已经超越了普通人的表现。测试涵盖替代用途任务、远距联想测试等多个维度。这不仅刷新了人们对 AI 能力的认知，也引发了关于创造性工作未来的深层讨论。

AI 驱动的 Pro Se 诉讼案件激增 社会

由于 AI 工具大幅提升了法律文书的质量，自我代理（Pro Se）的诉讼案件数量正在显著上升，部分地区法院报告民事案件增长了 20-40%。AI 正在让普通人也能写出像样的法律文件，这对司法系统既是机遇也是挑战。

模型与产品

NVIDIA Nemotron 3 Nano Omni：30B MoE 开源多模态 Agent 模型 模型

NVIDIA 发布 Nemotron 3 Nano Omni，采用 30B 总参数 / 3B 活跃参数的混合专家架构（MoE），统一处理视觉、音频、图像和文字。核心优势：比同类开源全模态模型高 9x 吞吐量，2.9x 单流推理速度。在文档智能、视频理解、音频理解六项 Leaderboard 上排名第一。

部署灵活性是一大亮点：从 Jetson 边缘设备、DGX Spark/Station 到云端都能一致部署。已被 Foxconn、Palantir、Docusign、Oracle 等采用或评估。权重、数据集和训练配方全部开源，可通过 HuggingFace、OpenRouter 和 NVIDIA NIM 微服务获取。

DeepSeek V4-Pro 永久降价 75%，V4.1 定档 6 月 降价

DeepSeek 宣布 5 月 31 日起 V4-Pro API 永久降至原价 1/4：缓存命中输入 0.025 元/百万 token，缓存未命中输入 3 元/百万 token，输出 6 元/百万 token。此前的 2.5 倍折扣已转为永久定价。

同时，DeepSeek 完成了 500 亿元人民币首轮融资（~$7.35B），创中国 AI 单轮融资纪录。投资方包括国家集成电路产业基金、腾讯、阿里等。投后估值超 3500 亿元（~$51.5B），3 周内估值从 $10B 翻 5 倍。V4.1 版本将支持原生 MCP 协议和图像/音频多模态输入。

阿里 Qwen3.7-Max：Agent 时代的国产旗舰 模型

阿里巴巴在云峰会发布 Qwen3.7-Max，从 Qwen 3.5 -> 3.6 -> 3.7 快速迭代。核心能力：1M token 上下文、35 小时自主复杂任务执行、约 10x 推理速度提升。在 Arena 全球盲测中位居国产第一，超越 Kimi K2.6、DeepSeek V4-Pro、GLM-5.1，逼近 GPT-5.5 和 Claude Opus 4.7。

这标志着国产大模型已经从"能力追赶"阶段进入"场景竞争"阶段，长时间自主 Agent 执行能力成为新的差异化方向。

面壁智能 BitCPM-CANN：国产昇腾训练的 1.58-bit 端侧大模型 开源

面壁智能联合清华大学、OpenBMB 社区开源 BitCPM-CANN，中国首个端到端基于华为昇腾芯片训练的 1.58-bit（三元量化）大模型。提供 0.5B/1B/3B/8B 四个参数规格。

技术亮点：相比 BF16 精度约 6x 内存节省，保留 90-97.2% 能力。基于 MindSpeed x Megatron-LM 训练栈，支持 32K 长序列。8B 版本可在主流旗舰手机上运行。已在 HuggingFace 和 ModelScope 开源。这对构建独立于 NVIDIA 的国产 AI 芯片生态具有战略意义。

Gemini 3.5 Pro 预计 6 月发布 预告

Google 在 I/O 2026 上确认 Gemini 3.5 Pro 将在 6 月发布，目标是缩小与 Claude Opus 4.7 和 GPT-5.5 在推理和长上下文任务上的差距。参考 Gemini 3.5 Flash 的表现（MMMU-Pro 84%、Terminal-Bench 76.2%、MCP Atlas 83.6%），Pro 版本的 GPQA Diamond 目标预计在 85% 以上。预估定价 $2.50/$15 per M tokens。

企业与 Agent

Microsoft Copilot Studio：电脑操作 Agent 和实时语音 Agent 双双 GA Agent

5 月 26 日更新中，Microsoft Copilot Studio 两大 Agent 能力正式 GA。电脑操作 Agent（Computer-Using Agent）5 月 13 日上线全球所有 Power Platform 商业区域，搭载 OpenAI CUA 和 Claude Sonnet 4.5 模型，支持 Azure Key Vault 凭据存储、Microsoft Purview 审计日志和可配置的人机协作审核。

实时语音 Agent 在北美通过 Dynamics 365 Contact Center 正式可用，响应延迟低于 500ms，支持语音到语音的实时对话。新的编排层将评估性能提升约 20%，同时减少 50% 的 token 消耗。未来路线图包括 Agent 通过观察用户操作来学习任务、双向语音管道实现实时人机协作。

Gemini API 重大变更：6 月 8 日旧接口将被移除 API

5 月 26 日，Gemini API 的 Interactions API schema 发生了重大变更，新的 request/response schema（outputs 和 steps 字段）以及 output format configuration（response_format）已成为默认。旧版 schema 将在 6 月 8 日完全移除，未迁移的应用将直接中断。两周硬性迁移窗口。

Microsoft Build 2026 前瞻：多模型策略与 Agent 协作 大会

6 月 2-3 日旧金山。预计发布：Azure AI Foundry 正式支持 Anthropic Claude（不再 OpenAI 独占）；GitHub Copilot 多 Agent 编排；AI Foundry for Windows SDK；Agent 365 治理审计能力。这标志着微软从"All-in OpenAI"转向多供应商前沿模型策略，降低单一供应商风险。紧接其后是 6 月 8 日的 Apple WWDC。
